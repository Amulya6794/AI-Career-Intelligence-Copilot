# AI Career Intelligence Copilot

An AI-powered career assistant built using Python and Streamlit.

## Features

* Resume PDF Parsing
* Skill Extraction
* ATS Score Analysis
* Resume Feedback
* Job Description Matching
* Skill Gap Analysis
* Career Roadmap Generation
* Profile Strength Assessment
